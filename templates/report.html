1. Introduction

SocialPulse Analytics is an AI-powered social media intelligence platform that ingests posts from X (Twitter) and Facebook (via the Apify scraping API), runs twelve analytical modules spanning NLP, network science, and machine learning, and renders the results as interactive dashboards. This document is the automated report for case {{ case.name }}.

2. Problem Statement

Modern brands need to understand audience sentiment, identify influencers and communities, detect misinformation, optimize ad spend, and forecast which content will perform — all in one unified workspace. SocialPulse implements all twelve required analytics modules and exposes them through per-case dashboards and exportable reports.

4. System Architecture

  • Frontend: Flask templates with Bootstrap 5, Chart.js, and D3.js
  • Backend: Python 3 + Flask + Flask-Login + Flask-SQLAlchemy
  • Database: SQLite (zero-config; switchable to MySQL/PostgreSQL via DATABASE_URL)
  • Data Source: Apify Twitter / Facebook actors with synthetic-data fallback
  • NLP / ML: VADER, scikit-learn (TF-IDF, Logistic Regression, Gradient Boosting, KMeans), NetworkX
  • Reporting: Jinja2 + xhtml2pdf for HTML/PDF export
